XML(Extensible Markup Language)

A worldwide used system for defining data formats. It provides a system to self made tags and complex documents.In other words we can say that XML allows the users to create data of information without having the data's pre-existing structure in the file.

XML is usually used in various Web applications and also XML documents are used by some programs

XML points:-

  1. XML is easy to generate.
  2. Encoding structure/documents of XML are in human-friendly format.
  3. XML has extension ending with (.xml).
  4. With an independent data format it is supported by many programming languages and frameworks

Structure of XML document

Tree structure

Example and explanation

  1. The first line is the XML declaration.
  2. The next line describes the root element of the document(<note>).
  3. The next 4 lines describe 4 child elements of the root (to, from, heading, and body).
      <to>Prafull</to>
     <from>Gururani</from>
     <heading>Btech</heading>
     <body>A computer science student</body>
  4. Final line defines the end of root element(</note>)
